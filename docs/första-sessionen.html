Kapitel 3 Första sessionen

Starta RStudio. Välj File > New File > R Script. Ett nytt fönster (Untitled1) öppnas.

Detta är ett script-fönster eller en editor. I scriptet (editorn) skrivs kommandon som utvärderas av R som sedan redovisar resultaten i R Console. Det är möjligt att skriva kod direkt i R Console, men rekommenderas inte. En script-fil är en textfil (med tillägget .R) med kommandon. Det är viktigt att spara script eftersom det är önskvärt att inte förlora texten. Skapa en mapp som heter A5Rkod. Välj File > Save As… och spara i A5Rkod scriptet Untitled med namnet myfirstscript.R.

Kommandon i scriptfilen körs (exekveras) på följande vis:

  1. En rad: Raden där markören är placerad körs med Ctrl+Enter (command + Enter på Mac) eller Run ovanför scriptet.
  2. Flera rader: Markera kodavsnittet och tryck Ctrl+Enter (command + Enter på Mac) eller Run.
  3. Hela scriptet: Ctrl+Shift+Enter

En rad i ett script måste avslutas med ett räknetecken \((+,-,*,/)\) om avsikten är att köra kod som sträcker sig över flera rader. En rad inleds aldrig med räknetecken. En rad som inleds med # exekveras inte och # används för att kommentera koden. Att kommentera kod är viktigt eftersom det förenklar för andra och dig själv att förstå vad koden innebär

  1. Skriv in nedanstående script till myfirstscript.R. Spara.
# myfirstscript. 
#  R som miniräknare
1+1
1+3
2-7
2*3
4/5
3^2
  1. Använd R som miniräknare genom att prova alla tre sätt att köra kod. Det är nödvändigt att bekanta sig med hur kod körs för att bli bekväm med att simultant arbeta i script-fönstret och se resultat i Console.

  1. Matematiska funktioner, t ex kvadratroten, finns implementerade i R. Skriv sqrt(6) i scriptet och spara. Kör koden och erhåll följande i Console.

  1. Även exponentialfunktionen \(\exp(x)\) finns i R. Skriv exp(3) i script-filen, spara och kör.

  1. Konstanten \(\pi\) finns i R. Skriv pi i script-filen, spara och kör.

  1. Kod finns ofta på flera rader. Anta att vi önskar beräkna \(2 + 2+ 3+ 5\) men att koden inte får plats på en rad utan måste delas upp på två rader. Skriv in följande rader i scriptet, spara, markera bägge raderna och kör.
# Kod över 2 rader
2 + 2 + 
3 + 5

Följande resultat ska presenteras i Console.

# Kod över 2 rader

Nu utvärderas raderna var för sig, vilket inte var avsikten med analysen.
